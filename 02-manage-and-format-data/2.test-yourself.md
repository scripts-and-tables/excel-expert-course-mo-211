## Part 3 â€” Hands-On Solutions

**T1.** Select the Amount column â†’ `Ctrl + 1` â†’ **Number** tab â†’ **Custom** â†’ in the **Type** field enter:

2. Click `B2` on `Form` â†’ **Data > Data Validation** â†’ **Allow: List** â†’ **Source:** select the row of country labels.
3. Click `C2` â†’ **Data > Data Validation** â†’ **Allow: List** â†’ **Source:** `=INDIRECT(B2)` â†’ **OK**. Excel will warn that the source evaluates to an error if `B2` is currently empty â€” accept and move on.

**T3.**

2. **Table Design** > **Table Name** â†’ type `Orders` â†’ `Enter`.
3. **Table Design** > tick **Total Row** â†’ click the Revenue total cell â†’ pick **Sum** from the dropdown.
4. With a cell inside `Orders` selected, **Insert > PivotTable** â†’ New Worksheet â†’ **OK**. Drag `Region` to **Rows** and `Revenue` to **Values**. The PivotTable's source is now `Orders` â€” adding rows to the Table will be picked up on **Refresh** (`Alt + F5`).

**T4.**
1. Click anywhere in the data â†’ **Data > Sort** â†’ sort by `Department` ascending.
2. **Data > Outline > Subtotal**.
3. **At each change in:** Department. **Use function:** Sum. **Add subtotal to:** tick `Amount`. Click **OK**.

**T5.**
1. Build a criteria range somewhere out of the way, e.g., `K1:L3`:

| Region | Sales |

4. Choose **Copy to another location**.

**T6.**
1. Select `A2:E100`.
2. **Home > Conditional Formatting > New Rule...**
3. Pick **Use a formula to determine which cells to format**.

**Exam Tip:** When the question wording mentions *"automatically include new rows,"* *"summary that responds to filtering,"* or *"named, named-column data,"* the intended tool is almost always an **Excel Table** (`Ctrl + T`). When the question says *"OR criteria,"* *"copy results elsewhere,"* or *"unique records only,"* the intended tool is **Advanced Filter**.
