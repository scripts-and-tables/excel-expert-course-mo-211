### Multiple Choice

| Question | Correct Answer | Explanation |
| :--- | :--- | :--- |
| **Q1** | **B** | Pareto = descending bars + cumulative percentage line on a secondary axis. Both charts live under `Insert > Insert Statistic Chart`. |
| **Q2** | **C** | **Overflow bin** collapses everything above the threshold into a single right-most bar (`>N`). Underflow does the same on the low side. |
| **Q3** | **B** | Funnel charts expect a **single value column with category labels**. Multiple value columns confuse the renderer. |
| **Q4** | **B** | The two-click sequence selects a single data point; **Set as Total** seats it on the baseline. |
| **Q5** | **B** | **Defer Layout Update** suspends recalculation until you click **Update**, ideal for large models. |
| **Q6** | **C** | The toggle lives in the **Options** dropdown of the `PivotTable Analyze` tab (or in `File > Options > Formulas`). |
| **Q7** | **B** | Double-click any value cell to spawn a new sheet containing the underlying source rows (drill-through / Show Details). |
| **Q8** | **C** | A Timeline gives a continuous scrubber with Years/Quarters/Months/Days; a date Slicer would list every unique date as a button. |
| **Q9** | **C** | **Same for All Sparklines** forces a shared min/max so magnitude differences become visible. |
| **Q10** | **B** | Grouped sparklines share styling. **Ungroup** first, then recolor the individual sparkline. |
| **Q11** | **C** | Accelerating growth = **Exponential** (provided all values are positive). Linear understates the recent doubling. |
| **Q12** | **B** | Tick **Display R-squared value on chart** in the `Format Trendline` pane. |
| **Q13** | **B** | **Forward = 3** projects the trendline 3 periods past the last data point. **Backward** projects before the first point. |
| **Q14** | **C** | Calculated Fields support only basic math on existing PivotTable fields. **No cell references, no VLOOKUP/SUMIF, no defined names.** |

### True/False

| Question | Answer | Explanation |
| :--- | :--- | :--- |
| **Q15** | **True** | A funnel renders top-to-bottom in source order. Largest-first gives the standard tapered shape. |
| **Q16** | **False** | **Defer Layout Update** is a per-session toggle and is reset when you close the workbook. |
| **Q17** | **True** | Timelines support **Report Connections** identically to Slicers, provided the PivotTables share the same data cache. |
| **Q18** | **False** | Win/Loss only encodes the **sign** of each value. A `+1` bar is the same height as a `+1,000,000` bar. |
| **Q19** | **False** | Higher-order polynomials almost always overfit and project wildly outside the data range. Higher R² on training data is not a forecast guarantee. |
| **Q20** | **True** | Show Details respects all active filters, slicers, and timelines — the drill-through sheet matches the visible slice. |
